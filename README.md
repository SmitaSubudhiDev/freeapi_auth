# FreeAPI Auth

- Secure login system
- Session management
- Password handling

- Responsive CSS styling
- Mobile-first approach
- Smooth transitions and animations
- Professional color scheme

📱 **Responsive Layout**
- Works seamlessly on desktop, tablet, and mobile devices

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SmitaSubudhiDev/freeapi_auth.git
```

2. Navigate to the project directory:
```bash
cd freeapi_auth
```

3. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

Alternatively, you can open the file directly by double-clicking `index.html`.

## Usage

### Login
1. Navigate to the login section
2. Enter your username and password
3. Click the "Login" button
4. Upon successful authentication, you'll be redirected to the dashboard

### Registration
1. Navigate to the registration section
2. Fill in the required fields (username, email, password,)
3. Click the "Register" button
4. Your account will be created and you can proceed to login
